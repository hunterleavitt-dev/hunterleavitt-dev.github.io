OpenFarm

An open-source platform for managing groups of 3D printers, developed in collaboration with Joshua Schell, Tyler Peters, and Matthew Elliott.

Overview

OpenFarm is an open-source platform designed to simplify the management and monitoring of multiple 3D printers. As one of the lead developers and principal maintainers of this project, I've contributed to building a comprehensive solution that enables users to efficiently manage printer farms, monitor print jobs, and coordinate resources across multiple devices.

Technologies Used

  • C#
  • Python
  • RabbitMQ
  • OctoPrint
  • Avalonia UI
  • Google Forms
  • Postgres
  • Docker
  • MinIO

Features

  • Centralized management of multiple 3D printers
  • Real-time monitoring of print jobs and printer status
  • Job queue management and scheduling
  • Resource allocation and optimization
  • Open-source architecture for community contributions
